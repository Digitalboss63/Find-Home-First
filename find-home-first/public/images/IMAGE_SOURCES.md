# Image Sources — Find Home First

All images stored in this directory are used to provide contextual neighborhood
photography in the Housing Search feature. They are not decorative — they help
housing coordinators identify the character of each community.

## Usage Guidelines

- Images are sourced from Unsplash and are free for use under the Unsplash License.
- All images must have descriptive alt text describing the neighborhood character.
- No image is used purely decoratively — each maps to a named community in the data model.
- When switching to real data, replace with photography specific to actual properties.
- Maintain consistent 16:10 aspect ratio (800×500px) for all community images.
